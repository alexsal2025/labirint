# Решатель лабиринтов

GUI-программа для Windows/Python:

- открывает фото или скан лабиринта;
- пытается сама найти вход и выход по краям;
- если авто-режим не справился, можно кликнуть старт и финиш вручную;
- рисует найденный путь прямо поверх изображения;
- умеет сохранить готовый результат.

## Запуск

Самый простой вариант:

```bat
run_maze_solver.bat
```

Или через Python:

```powershell
python maze_solver_gui.py
```

## Зависимости

```powershell
pip install -r requirements.txt
```

## Как пользоваться

1. Нажми `Открыть изображение`.
2. Если у лабиринта есть явный вход и выход по краям, просто нажми `Найти путь`.
3. Если автоопределение не сработало, кликни на старт и финиш на картинке, затем снова нажми `Найти путь`.
4. Сохрани результат кнопкой `Сохранить результат`.

## Что лучше всего подходит

Лучше всего работают изображения, где:

- стены заметно темнее или светлее проходов;
- вход и выход находятся на внешней границе лабиринта;
- фото не слишком размыто.
